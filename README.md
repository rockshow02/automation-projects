## 📂 Demo Projects

### 1. 📧 Email Report Otomatis dari Google Sheets

**Problem:** Tim finance menghabiskan 2 jam/hari untuk merekap data dan mengirim laporan manual.  
**Solution:** Workflow n8n yang berjalan otomatis setiap pagi — ambil data dari Google Sheets, format jadi laporan, kirim via Gmail.  
**Result:** 0 menit kerja manual. Laporan terkirim tepat pukul 08.00 setiap hari.

```
Google Sheets → n8n Scheduler → Format Data → Gmail
```

### 2. 💬 Notifikasi WhatsApp Otomatis

**Problem:** Owner bisnis tidak tahu ada order/pendaftaran baru kecuali cek manual.  
**Solution:** Setiap ada entri baru di Google Form → notifikasi WhatsApp langsung ke HP owner dalam hitungan detik.  
**Result:** Respons pelanggan lebih cepat, tidak ada order yang terlewat.

```
Google Forms → Make.com → WhatsApp (CallMeBot)
```
